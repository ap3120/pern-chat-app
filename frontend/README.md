# Getting Started with PERN chat App Frontend

This project is a the React frontend of a chat application.

## Prerequisites
You need to have the following installed:
1. Nodejs
2. Node package manager

## Before starting

### Environment variables
To use the application create a `.env` file following the `.env-sample`.

### Dependencies
`npm install`

## Starting the frontend

1. To run inside electron: `npm run dev`
2. To run in the browser: `npm start`

## Testing
1. Run in headless mode: `npx playwright test`
2. Run in ui mode: `npx playwright test --ui`</br>
The report is generated in `playwright-report/index.html`
